Bagaimana Melakukan Setup Play-1.2.7.2-java8 di Eclipse? (by Andik)

  1. Create empty project.
  2. In Project > Properties > Libraries, add jars in framework/jar/*.jar and modules\testrunner\lib\*.jar
  3. To run an application, add -Dapplication.path=PATH_TO_APP in eclipse launcher.
  4. To run in test mode, add -Dplay.id=test
  5. Prepare a test application located in framework/test-src. Make suret that folder in include as 'source folder' of project
